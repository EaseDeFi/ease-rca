# Reciprocally-Covered Assets

## Local deployment
1. clone this repo 
```git clone git@github.com:ArmorFi/armor-rca.git```
2. create new `.env` file and fill values from `.env.example`
3. Compile contracts - `npm run build`
4. Run tests -  `npm test`
